üéØ Designer Checklist

Things to keep in mind when designing for IEEE Platform

‚úÖ Do's

  • Use Poppins font for all text elements
  • Apply primary teal (#17a697) for interactive elements
  • Use 135¬∞ gradients consistently
  • Maintain 1.5rem gaps between grid elements
  • Add lift effect on hover: translateY(-2px)
  • Always include focus states for accessibility
  • Use line-height 1.6 for body text readability
  • Test color contrast ratios (WCAG AA minimum)

‚ùå Don'ts

  • Mix gradient angles (keep 135¬∞)
  • Use more than 2 colors in a gradient
  • Set font-size below 0.75rem
  • Use pure black (#000) for text
  • Add shadows without purpose
  • Skip focus states for keyboard navigation
  • Mix typefaces (use Poppins only)
  • Ignore mobile responsiveness

üì± Responsive Breakpoints to Remember

Desktop: 1200px+ ‚Äî Full feature set, multi-column layouts
